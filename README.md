# Real-Time Object Detection using MobileNet SSD and OpenCV

## Project Overview

This project performs real-time object detection using a webcam feed. It uses the MobileNet SSD deep learning model along with OpenCV to detect and classify objects, displaying bounding boxes and confidence scores.

## Technologies Used

* Python
* OpenCV
* NumPy
* Imutils
* MobileNet SSD
* Caffe Model

## Features

* Real-time object detection
* Webcam-based video processing
* Bounding box visualization
* Confidence score display
* Multiple object recognition
