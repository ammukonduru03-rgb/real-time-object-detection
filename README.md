# Real-Time Object Detection using MobileNet SSD and OpenCV

## Project Overview

This project performs real-time object detection using a webcam feed. It uses the MobileNet SSD deep learning model along with OpenCV to detect and classify objects, displaying bounding boxes and confidence scores.

## Technologies Used

* Python
* OpenCV
* NumPy
* Imutils
* MobileNet SSD
* Caffe Model

## Features

* Real-time object detection
* Webcam-based video processing
* Bounding box visualization
* Confidence score display
* Multiple object recognition

## Files

* object_detection.py
* MobileNetSSD_deploy.prototxt.txt
* MobileNetSSD_deploy.caffemodel

## Model File

The MobileNetSSD_deploy.caffemodel file is not included in this repository due to GitHub upload limitations.

To run this project, download the pretrained MobileNet SSD Caffe model and place:

MobileNetSSD_deploy.caffemodel

in the project directory alongside the Python script.
